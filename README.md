# 微控制器(Microcontroller)應用

## 微控制器是一種集成在單一晶片上的小型電腦，它包括處理器核心、記憶體和可程式輸入/輸出周邊。微控制器廣泛用於自動控制系統，如自動化設備、汽車電子設備、手持設備等。
